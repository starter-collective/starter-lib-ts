# 快速开始

## 安装

::: code-group

```sh [npm]
npm install starter-ts
```

```sh [yarn]
yarn add starter-ts
```

```sh [pnpm]
pnpm install starter-ts
```
:::

## CDN

```js
<script src="https://unpkg.com/starter-ts"></script>
```

它将作为 `window.StarterLibTs` 暴露在全局对象中

## 使用示例

```js
import { sayHello } from 'starter-ts'

sayHello('Kieran')
```

更多详情请参考[函数列表](/functions/index)。
